# 快速开始

本文主要介绍怎样快速开始使用react-umeditor富文本编辑器。

## 安装

首先，选择安装react-umeditor到你的项目中：

	npm install --save react-umeditor
	
## 使用

根据不同使用的场景，你应该选择以下不同的场景阅读相应的文档。
